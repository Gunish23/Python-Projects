# 📊 E-commerce Sales Dashboard (Pandas + NumPy + Matplotlib)

This is a simple data analysis project where I created a complete e-commerce sales dashboard using only **Pandas, NumPy, and Matplotlib**.

The goal of this project was to practice data analysis by generating a dataset, cleaning it, analyzing it, and visualizing useful business insights — all in one script.

---

## 💡 What this project does

* Generates a random e-commerce dataset (orders, customers, products)
* Performs data analysis using Pandas
* Calculates important business metrics
* Identifies best-performing products and categories
* Shows monthly sales trends
* Visualizes data using Matplotlib charts

---

## 📈 Key Insights Generated

* Total Revenue
* Total Orders
* Average Order Value
* Best-selling Product
* Best-selling Category
* Top 5 Products by Revenue
* Category-wise Performance
* Monthly Revenue & Order Trends

---

## 📊 Visualizations Included

* Bar chart → Top 5 products by revenue
* Pie chart → Revenue by category
* Line chart → Monthly revenue trend
* Bar chart → Monthly orders

---

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib

---

## ▶️ How to run

1. Install required libraries:

```bash
pip install pandas numpy matplotlib
```

2. Run the script:

```bash
python ecommerce_dashboard.py
```
