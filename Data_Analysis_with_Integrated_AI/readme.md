🤖 AI-Powered Data Cleaning & Analysis Studio (Tkinter)
An advanced AI-powered desktop application built using Python (Tkinter, Pandas, Matplotlib, Scikit-learn) that simplifies the complete data analysis workflow — from raw messy data to actionable insights.

This tool automates data cleaning, analysis, visualization, and intelligent insights generation, helping users save time and improve productivity.

🚀 Features
🔹 1. Smart Data Loading
Supports CSV & Excel files
Automatic dataset preview
Displays rows, columns, and structure instantly
🔹 2. AI-Based Auto Cleaning
Standardizes column names

Removes duplicate rows

Handles missing values (median/mode based filling)

Detects and converts:

Dates
Numeric values
Outlier detection using IQR method

Drops useless columns automatically

🔹 3. Manual Data Editing
Replace values column-wise (interactive UI)

Smart formatting:

Currency → Numeric
Add/remove prefixes
Title case conversion
Date format conversion
AI suggestions for best formatting

🔹 4. Search, Filter & Sort
Real-time search across dataset

Advanced filtering:

Contains, equals, starts/ends with
Numeric conditions (>, <, ≥, ≤)
Sorting (ascending/descending)

🔹 5. Advanced Data Analysis
Column type detection (AI-based)
Missing & unique value analysis
Numerical statistics summary
Correlation matrix
Categorical insights (top values)
Date range insights
🔹 6. Machine Learning Insights
🔍 Anomaly Detection (Isolation Forest)
📊 Clustering (K-Means)
📈 Predictive Trend (Linear Regression with R² score)
🔹 7. Data Visualization
Supports multiple chart types:

Histogram
Bar Chart
Line Chart
Scatter Plot
Boxplot
Pie Chart
Correlation Heatmap
Outlier Detection Chart
🔹 8. AI Business Insights
Converts analysis into human-readable insights

Highlights:

Trends
Relationships
Key observations
Suggests next steps for decision-making

🔹 9. Export & Reset
Export cleaned data to:

CSV
Excel
Reset dataset to original state anytime
